<Info>
这是为找回“解释世界的眼睛”而制定的试行方案。每日坚持写 100 字，不求华丽辞藻，只求精准捕捉。
</Info>

| 天数 | 练习重点 | 任务描述 |
| :--- | :--- | :--- |
| **Day 1** | **唯一的特征** | 描写一个你熟悉的人，但不准写相貌。通过他一个习惯性的动作或声音，让人一眼认出他。 |
| **Day 2** | **光影的物理感** | 描写此刻照进你窗户的光。它是像水一样流淌，还是像针一样刺破空气？写出它的质感。 |
| **Day 3** | **声音的形状** | 闭上眼三分钟，记录你听到的一种最不起眼的声音（如冰箱嗡鸣、邻居脚步），赋予它生命力。 |
| **Day 4** | **物件的秘密** | 挑选桌上的一个杂物（打火机、水杯）。想象它如果不只是个工具，它此刻的情绪是怎样的。 |
| **Day 5** | **精准动词** | 描写一段动态（如倒茶、系鞋带）。禁用形容词，只用动词来表现动作的力量和节奏。 |
| **Day 6** | **陌生的街角** | 描写你每天经过的一处风景，强迫自己发现一个你过去一年从未注意到的细节。 |
| **Day 7** | **情绪的实体化** | 描写某种抽象的情绪（如焦虑或期待）。不要出现这个词，要通过具体的物象表现它。 |
