# Dog-Skills：48 个 Claude Code 技能库

> GitHub: [ZhouYinLong-lab/Dog-Skills](https://github.com/ZhouYinLong-lab/Dog-Skills) · v1.0.0 · MIT

过去两周，我在 Claude Code 的技能生态里疯狂探索——评估了上百个 skill，装了又卸，最后精选 **48 个**沉淀成一个仓库。覆盖范围远超编程：有帮你做品牌设计的、有费曼学习法教练、有体检报告解读、甚至还有让 Claude 唱完 bug 给你听的。

这篇文章逐一介绍每个技能，包括架构（元 skill 会画图）、安装方式和来源。

<Info>
**什么是 Claude Code Skill？** 本质是一个 `SKILL.md` 文件，里面写了触发关键词和指令流程。放在 `~/.claude/skills/` 下，Claude Code 会自动识别并在匹配时调用。
</Info>

## 🧠 思考与研究（6 个）

### `first-principles` — 第一性原理思考
> 强制 AI 剥离所有假设，从最基本的事实重新推导。适用于修 BUG、架构设计、商业决策。

**安装**：`cp -r first-principles ~/.claude/skills/first-principles`

---

### `storm-research` — 斯坦福 STORM 深度研究
> 4 个提示让 Claude 像博士一样做研究。5 分钟产出博士 48 小时级别的研究简报。

**安装**：`cp -r storm-research ~/.claude/skills/storm-research`

---

### `feynman-learning` — 费曼学习法 AI 助手
> 四步循环——教给别人→发现缺口→回归原始→简化类比——20 分钟深度理解任意概念。

**安装**：`cp -r feynman-learning ~/.claude/skills/feynman-learning`

---

### `nuwa` — 女娲·人物思维蒸馏
> 输入人名，自动深度调研→提炼思维框架→生成可运行的 AI Skill。已蒸馏 Paul Graham、张一鸣、Karpathy 等。

**安装**：`cp -r nuwa-skill ~/.claude/skills/nuwa`

---

### `last30days` — 社交媒体话题调研
> 跨 Reddit/X/YouTube/TikTok/HN/Polymarket 等 15+ 平台，按真实互动量（而非 SEO）排序。输入 `/last30days <话题>`。

**安装**：`cp -r last30days-skill ~/.claude/skills/last30days`

---

### `thinking-toolkit` ⭐ 元技能

> **架构**：整合 6 大思考框架为一个入口。

```
thinking-toolkit
├── superforecaster     ← 五阶段校准预测（基准率→证据→置信区间）
├── decision-matrix     ← 多维度量化决策矩阵+敏感度分析
├── bayesian-reasoning  ← 基于新证据持续更新信念

**安装**：`cp -r thinking-toolkit ~/.claude/skills/thinking-toolkit`

```
