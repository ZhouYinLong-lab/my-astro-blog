### 🪶 夜鹭心情识别器

这是一个结合了自然语言处理(NLP)和纯前端 CSS 动画的互动小实验.它通过 Vercel 的 Serverless Function 安全连接了最新的 Gemini 大模型,能够精准读懂你字里行间的情绪!

### 🛠️ 技术栈揭秘
* **前端骨架**:纯 HTML + CSS + 原生 JavaScript
* **交互动效**:CSS `@keyframes` 结合 JS DOM 重排触发果冻回弹动画
* **后端大脑**:Vercel Serverless Functions (Node.js) 隐藏 API Key 并处理跨域
* **AI 模型**:`gemini-2.5-flash` 极速语义识别
