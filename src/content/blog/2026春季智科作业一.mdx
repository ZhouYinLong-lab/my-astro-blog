# C++ 核心机制与面向对象实战 (一)

在 C++ 的学习过程中，理解底层的内存管理、编译机制以及面向对象的基础设计模式至关重要。本文将对近期梳理的几个核心知识点进行系统性总结。

---

## 1. 深入剖析 C++ 程序的四大内存分区

<Info>
**关键概念**：程序运行时的内存被划分为四个区域，每个区域都有其特定的生命周期和管理方式。
</Info>

* **代码区（Code Segment）**：存放机器指令，通常是只读的。
* **全局/静态存储区**：存放全局变量和 `static` 变量。
* **栈区（Stack）**：由编译器自动分配释放，存放局部变量。
* **堆区（Heap）**：由程序员通过 `new`/`delete` 手动管理。

---

## 2. 预处理宏定义的陷阱

<Error>
**注意优先级**：如果宏定义内部缺少括号保护，在复杂运算中会引发逻辑错误。
</Error>

<Diff>
```diff
- #define max_val(x,y) ((x)>(y))?(x):(y)
+ #define max_val(x,y) (((x)>(y))?(x):(y))
```
</Diff>

例如在执行 `max_val(10, 5) + 20` 时，由于 `+` 优先级更高，不加外层括号会导致结果变为 10 而非预期的 30。

## 4. 性能优化：inline 内联函数

内联机制通过将函数代码直接嵌入调用处，消除了函数调用的开销。这在处理高频调用的小规模函数时，是性能优化的利器。

---

## 5. 动态内存管理：new vs malloc

| 特性 | `new` / `delete` | `malloc` / `free` |
| :--- | :--- | :--- |
| **属性** | 运算符 | 库函数 |
| **对象管理** | 自动调用构造/析构函数 | 仅分配/释放内存 |
| **类型安全** | 返回确切类型指针 | 返回 `void*`，需强转 |

## 6. 面向对象实战：Stack 的两种实现

针对栈（Stack）这一经典结构，我们分别探讨**动态数组**与**链表**两种底层实现。

### 方案一：基于动态数组的扩容栈
利用连续内存，支持快速随机访问，通过容量翻倍机制处理溢出。

```cpp
class ArrayStack {
private:
    int* data;
    size_t capacity, topIndex;
    
    void expand() {
        size_t newCap = (capacity == 0) ? 1 : capacity * 2;
        int* newData = new int[newCap];
        for (size_t i = 0; i < topIndex; ++i) newData[i] = data[i];
        delete[] data;
        data = newData;
        capacity = newCap;
    }
public:
    ArrayStack() : data(nullptr), capacity(0), topIndex(0) {}
    ~ArrayStack() { delete[] data; }
    void push(int x) { if (topIndex == capacity) expand(); data[topIndex++] = x; }
    void pop() { if (topIndex > 0) topIndex--; }
    int top() const { return data[topIndex - 1]; }
};
```

### 方案二：基于链表的动态栈
通过指针连接节点，天然支持动态增长，操作复杂度稳定在 O(1)。

```cpp
struct Node {
    int val;
    Node* next;
    Node(int v) : val(v), next(nullptr) {}
};

class LinkedStack {
private:
    Node* head;
public:
    LinkedStack() : head(nullptr) {}
    ~LinkedStack() { while(head) pop(); }
    void push(int x) {
        Node* n = new Node(x);
        n->next = head;
        head = n;
    }
    void pop() {
        if (!head) return;
        Node* t = head;
        head = head->next;
        delete t;
    }
    int top() const { return head->val; }
};
```
