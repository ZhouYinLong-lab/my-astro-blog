# Contributing

Thank you for your interest in contributing! 

## How to Contribute

1. **Fork** the repository
2. **Create a branch** for your feature/fix: `git checkout -b feature/your-feature`
3. **Make your changes** and commit: `git commit -m "feat: describe your change"`
4. **Push** to your fork: `git push origin feature/your-feature`
5. Open a **Pull Request** with a clear description

## Guidelines

- Follow existing code style and conventions
- Keep pull requests focused on a single change
- If adding features, consider opening an issue first to discuss

## Questions?

Feel free to open an issue for questions or suggestions.
